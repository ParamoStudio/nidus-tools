How it works

  1. Browse or search for a tool, micro tool, or template.
  2. Download the file — it's a single .js (tools, micro tools) or .md (templates).
  3. In Nidus, open the Tool Library and choose Import — no auto-install, just pick the file you downloaded.
  4. Want to contribute your own? See CONTRIBUTING.md on GitHub.
