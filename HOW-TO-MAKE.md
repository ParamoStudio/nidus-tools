# How to make a Nidus add-on

You can make a **tool**, **micro tool**, or **template** for Nidus — even if you don't write code. The
easiest way is to let an AI build the file for you.

## The 3-step way (no coding)

1. **Pick what you want to make:**
   - **Tool** — a whole new tool in your workspace (its own tile, cards, and open view). *Example: a
     glaze recipe book.*
   - **Micro tool** — a small form that turns your input into a clean, formatted block you can paste
     anywhere. *Example: a unit converter.*
   - **Template** — a ready-made layout for a project's "Current Blueprint", tuned to a discipline.
     *Example: a woodworking piece.*

2. **Download that type's "skill" file:**
   - [Tool skill](skills/nidus-tool-skill.md)
   - [Micro tool skill](skills/nidus-microtool-skill.md)
   - [Template skill](skills/nidus-template-skill.md)

3. **Hand it to your favourite AI** (Claude, ChatGPT, or any other). Paste the skill file, then describe
   what you want to build in plain words — *"make me a tool that tracks my sourdough bakes"*. The AI
   follows the skill and writes the finished file. Save it.

That's it — you now have your add-on file, ready to use or share.

> **Prefer to do it by hand?** Each skill file is also a complete, human-readable spec. Read it and write
> the file yourself.

## Try it in Nidus first

Import your file into Nidus (**Tool Library → Import**) and make sure it works the way you want before
sharing it.

## Share it with others

Happy with it? Publish it on this marketplace so other makers can find it — see
[CONTRIBUTING.md](CONTRIBUTING.md). It's short, and an AI can do most of it for you.
